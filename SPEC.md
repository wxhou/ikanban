# 项目目标
管理一个项目下多个事件、多个任务，让老板能看到项目全局状况的看板网站。

## 用户角色
- 管理员/项目经理：创建项目、分配任务、管理成员
- 老板/管理层：只读视角，查看全局看板，了解各项目/任务的进度

## 核心功能
- [x] 项目列表
- [x] 看板视图（待办 / 进行中 / 已完成 等状态列）
- [x] 任务卡片（标题、负责人、截止日期、优先级）
- [x] 进度概览（老板视角的总览仪表盘）
- [x] 筛选和搜索
- [x] 拖拽任务移动状态
- [x] CSV/Excel 导出
- [x] 任务创建/编辑 Modal

## 设计约束
- 风格：Neutral Modern（已选定）
- 技能：Web Prototype（单 HTML 原型）
- 精度：高保真
